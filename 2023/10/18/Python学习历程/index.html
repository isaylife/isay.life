字符串

单个变量进行拼接

name = “字符串拼接”
message = “字符串拼接占位符 %s” % name

多个变量进行拼接

name1 = “字符串拼接”
name2 = “多个变量进行字符串拼接”
message = “字符串拼接占位符 %s” % (name1,name2)
%s为字符串占位（将内容转换成字符串）
%d为整数占位（将内容转换为整数）
%f为浮点型占位（将内容转换为浮点型）
//多个变量进行拼接需要按占位顺序来

数字精度控制

m控制宽度
.n控制小数点精度
%5d：整数占位5位
%5.2f:浮点数占位5位宽度，同时四舍五入精确小数点两位
%.2f：浮点数四舍五入保留后两位
